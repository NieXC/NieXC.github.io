Tools

Netscope[LINK]
Intro: A web-based tool for visualizing neural network architectures. It currently supports Caffe's prototxt format.

Arxiv Sanity Preserver[LINK]
Intro: This project is a web interface that attempts to tame the overwhelming flood of papers on Arxiv. It allows researchers to keep track of recent papers, search for papers, sort papers by similarity to any paper, see recent popular papers, to add papers to a personal library, and to get personalized recommendations of (new or old) Arxiv papers.
